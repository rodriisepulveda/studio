## :rocket: Quick Start

Sigue estos pasos para correr el proyecto en tu máquina local:

1. Clona el repositorio:
  ```bash
   git clone https://github.com/tu-usuario/pvs-solutions.git
   ```

2. Instala las dependencias:
  ```bash
  npm install
  ```

3. Inicia el servidor de desarrollo:
  ```bash
  npm run dev
  ```
